# SHIKII

Shikii is a programming practice from a programming learner. 



## I.  Introduction

Shikii is a product of the creator's programming practice in Javascript language.  It includes the adoption of React.js, Express.js and MongoDB in both frontend and backend development.

With the focus of stylish housewares, the project aims to create an platform with e-commerce and online marketing functions.  Common features of user authentication, dashboards management, product searching and filtering are included.
